PROCESS

How It Works

Four cryptographic steps stand between any file and an unforgeable proof of origin.

01

Upload File

Drop in any image, video, or document. CRYPTIQ never alters the original bytes.

02

Generate SHA-256 Fingerprint

A unique 256-bit hash is computed from the file's exact contents — a digital fingerprint.

03

Secure TEE Signature

The hash is sealed inside a Trusted Execution Environment and signed with a hardware-bound key.

04

Permanent Origin Certificate

An immutable, downloadable certificate is issued — proof of origin that travels with the file forever.
